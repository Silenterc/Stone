◆ play()

void Game::play ( bool  justLoaded = false)

The Game loop.

This is the main function of the Game. The Players exchange turns, draw Cards, execute commands and the Game evaluates it's state. Once it's finished, this function ends.

Parameters
justLoadedIf the Game just loaded, then it continues the round instead of creating a new round

◆ saveGame()

void Game::saveGame ( ) const

This function generates a file and Saves the game in the predetermined format into it.

The file's name is the local time. It writes the Game data (AI/PvP, playerTurn, isFirstRound) and calls both the Players's << operators, which print the Players' data into the file.
